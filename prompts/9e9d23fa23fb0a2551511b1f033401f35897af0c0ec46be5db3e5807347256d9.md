# 依赖管理

**原则**：使用包管理工具安装依赖，不直接修改依赖定义文件。

## 不同项目类型的包管理工具

- **Rust项目**：使用`cargo add/remove`
- **Solidity项目**：使用`forge install/remove`
- **前端项目**：使用`pnpm add/remove`（禁止使用npm）

## 示例

```bash
# 正确
cargo add tokio --features full
pnpm add wagmi

# 错误
# 手动编辑Cargo.toml或package.json
```
